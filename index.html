Fakebook

Fakebook is a Facebook clone complete with functionality for making posts, adding and searching for friends, and liking and commenting on friends' posts. It uses Rails on the backend, Javascript and React on the frontend, a PostgreSQL database, and a server running Webpack.

Pokemon Battle Simulator

This Pokemon Battle Simulator is a browser game implemented using vanilla Javascript and the HTML5 Canvas API. Two teams of six Pokemon are randomly generated and then players take turns either selecting a move with which to attack or a Pokemon to switch in. Play continues until all of one player's Pokemon have fainted, at which point a winner is declared!

MusicBox

MusicBox is an Amazon-themed online music store. Users can search for products in the database, add items to their cart, and they can also list products to be sold.
MusicBox uses GraphQL, React, Apollo, and Node. Backend data is stored using MongoDB as well as Amazon AWS S3. The application can be installed via Docker.

  • Github Repo

MeTube

MeTube is a Youtube clone and video database used for searching for, watching, saving, and uploading videos. It allows for user authentication and login, display of recently watched and trending videos, and interaction with other users through likes and comments. MeTube uses the MERN stack - MongoDB and Express on the backend and React and Node on the frontend.
